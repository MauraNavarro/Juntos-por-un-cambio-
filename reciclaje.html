Reciclaje creativo

  • Lámparas y jarrones de botellas de vidrio :Reciclar botellas de vidrio creando lámparas o jarrones es una idea popular que se puede hacer pintando las
    botellas con Chalk Paint, envolviéndolas en cordón para una apariencia más cálida o envolviéndolas con tela y encaje.
  • Reciclaje de ropa vieja : A la ropa que ya no se utiliza se le puede dar una segunda oportunidad creando bolsas para la compra, paños de cocina, rellenos
    para puffs o adornos para fiestas.
  • Reutilizar latas : Las latas se pueden utilizar para crear portalápices, maceteros u organizadores.
  • Crear arte a partir de papel : el papel se puede utilizar para crear una variedad de arte, como lámparas, juguetes, instrumentos musicales o decoraciones.
  • Reutilizar botellas de plástico : Las botellas de plástico se pueden utilizar como maceteros cortándolas por la mitad y haciendo un pequeño corte en la
    base para el drenaje. Esto puede contribuir a un modelo de consumo responsable dando una segunda o incluso una tercera vida a los materiales.
  • Utilizar cestas como papeleras de reciclaje en casa : Utilizar cestas como papeleras de reciclaje en casa puede ayudar a ordenar y clasificar materiales y
    productos que ya no se utilizan. Esto puede contribuir a un modelo de consumo responsable optimizando los recursos y consumiendo con conciencia.
